# 📊 Análisis de Recursos Humanos - Práctica 1

## 📝 Descripción del Proyecto
Este dashboard fue creado para analizar los indicadores clave (KPIs) del departamento de RRHH. El objetivo es identificar tendencias en la rotación de personal y la satisfacción laboral.

## 🛠️ Herramientas Utilizadas
* **Power BI Desktop**: Creación de visualizaciones y modelado.
* **Power Query**: Limpieza y transformación de datos (ETL).
* **DAX**: Creación de medidas personalizadas para calcular la tasa de rotación.

## 💡 Insights Principales
* Se observó que el **80% de las renuncias** provienen del área de ventas.
* El promedio de antigüedad antes de una promoción es de **2.5 años**.
